Halaman 1

BAB I: PENDAHULUAN

1.1 Latar Belakang

Coffee Senja merupakan sebuah sistem informasi manajemen coffee shop modern yang mengintegrasikan layanan cloud emulator untuk menangani data transaksi, antrean pesanan, dan penyimpanan struk digital. Proyek ini bertujuan untuk mensimulasikan lingkungan produksi cloud (AWS) di dalam infrastruktur lokal menggunakan LocalStack dan Docker.

1.2 Arsitektur Teknologi

Komponen Teknologi Keterangan
Bahasa Pemrograman Python 3.x Framework Flask sebagai logic utama.
Containerization Docker Compose Isolasi environment aplikasi dan service.
Cloud Emulator LocalStack Simulasi AWS DynamoDB, SQS, dan S3.
Database DynamoDB Penyimpanan User, Menu, dan Order.

1.3 Fitur Utama

  • Sistem Membership: Registrasi member dan riwayat pesanan pribadi.
  • Sistem Keranjang: Manajemen beberapa item dalam satu transaksi.
  • Sistem Promo: Penggunaan kode voucher untuk diskon otomatis.
  • Admin Dashboard: Monitoring pendapatan harian dan stok bahan baku.
Halaman 2

BAB II: PANDUAN INSTALASI

2.1 Persyaratan Perangkat Lunak

Sistem ini dirancang untuk portabilitas tinggi. Pastikan perangkat target memiliki:

  • Docker Desktop (untuk Windows/Mac) atau Docker Engine.
  • Browser modern (Chrome, Edge, atau Firefox).

2.2 Langkah-Langkah Deployment

Ikuti langkah berikut untuk menjalankan aplikasi di perangkat baru:

1. Persiapan Folder: Masuk ke direktori proyek melalui terminal.

cd coffee-senja-localstack

2. Menjalankan Container: Gunakan perintah build untuk menyusun environment cloud.

docker compose up --build

3. Verifikasi Koneksi:

  • Buka http://localhost:5000 untuk mengakses antarmuka web.
  • Buka http://localhost:4566 untuk memastikan LocalStack aktif.

2.3 Detail Alokasi Port

Service Port Fungsi
Flask Web App 5000 Interface Customer & Admin
LocalStack 4566 AWS Emulator Endpoint
Halaman 3

BAB III: PANDUAN PENGGUNAAN

3.1 Alur Customer & Member

Pengguna dapat memesan secara langsung atau mendaftar sebagai member:

  1. Registrasi: Klik menu Login, pilih tab "Daftar Member".
  2. Pemesanan: Pilih menu, tentukan jumlah/catatan, masukkan ke Keranjang.
  3. Checkout: Di dalam keranjang, gunakan kode promo (misal: SENJA20) untuk diskon, lalu klik Buat Pesanan.
  4. Riwayat: Khusus Member dapat melihat riwayat pesanan di menu Pesanan Saya.

3.2 Panduan Administrasi (Admin)

Akses admin digunakan untuk mengelola toko secara keseluruhan:

  • Dashboard Harian: Melihat pendapatan hari ini dan peringatan stok (Low Stock Alert).
  • Manajemen Menu: Admin dapat menambah menu baru dengan mengunggah foto, atau mengedit data menu yang sudah ada.
  • Proses Antrean: Menu "Admin Utama" digunakan untuk memproses pesanan masuk dari antrean SQS ke status READY.

3.3 Detail Kredensial

Akses Admin Default:

Username: admin | Password: admin123

Halaman 4

BAB IV: IMPLEMENTASI CLOUD & PENUTUP

4.1 Integrasi Cloud Emulator

Sistem ini mengimplementasikan tiga pola integrasi cloud AWS:

  • DynamoDB (Storage): Menggunakan skema kunci primer menu_id, order_id, dan username untuk akses data cepat.
  • SQS (Messaging): Pesanan yang dibuat akan masuk ke antrean asinkron, memastikan server tidak overload saat traffic tinggi.
  • S3 (Object Store): Setiap pesanan yang selesai akan menghasilkan file JSON struk digital yang disimpan secara permanen di S3 bucket.

4.2 Kesimpulan

Aplikasi Coffee Senja telah berhasil diimplementasikan sebagai solusi sistem informasi coffee shop yang handal. Penggunaan Docker dan LocalStack menjamin bahwa aplikasi dapat dideploy dengan mudah di berbagai perangkat tanpa perlu melakukan instalasi database secara manual di host sistem.
